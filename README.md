# Scientific Calculator

## Description
This Scientific Calculator project provides comprehensive functionalities for performing basic arithmetic operations as well as advanced calculations such as trigonometric and logarithmic functions. It aims to serve students, engineers, and anyone needing precise mathematical calculations.

## Features
- Basic operations: Addition, Subtraction, Multiplication, Division  
- Trigonometric functions: Sin, Cos, Tan  
- Logarithmic functions: Ln, Log  
- Graph plotting capabilities  

## Installation Instructions
1. Clone the repository:
   ```bash
git clone https://github.com/m-ali7787430000/Scientific-calculator.git
   ```
2. Navigate to the project directory:
   ```bash
cd Scientific-calculator
   ```
3. Install the required dependencies (if any):
   ```bash
npm install
   ```

## Usage Guide
- To start the application, run the following command:
  ```bash
npm start
  ```
- Follow the prompts on the interface to perform calculations.
