## Architecture
- The Scientific Calculator is designed following a modular architecture, which allows for easy updates and maintenance. Each module handles specific functionalities such as arithmetic operations, trigonometric functions, and logarithmic calculations.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/m-ali7787430000/Scientific-calculator.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd Scientific-calculator
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Run the project**:
   ```bash
   npm start
   ```
